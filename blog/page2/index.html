GNU ARM Eclipse News

GNU ARM Eclipse QEMU v2.7.0-20161128* released

Nov 28, 2016 - Version 2.7.0-201611282115 is a new release, adding support for active graphical buttons.

GNU ARM Eclipse Windows Build Tools v2.8-20161122* released

Nov 22, 2016 - Version 2.8-201611221915 is a maintenance release of the GNU ARM Eclipse Windows Build Tools.

GNU ARM Eclipse QEMU v2.7.0-20161029* released

Oct 29, 2016 - Version 2.7.0-201610290751 is a new stable release, following the offfical QEMU releases.

GNU ARM Eclipse OpenOCD v0.10.0-20161028*-dev released

Oct 28, 2016 - Version v0.10.0-201610281609-dev is a maintenance release. It includes binaries for Windows, macOS and GNU/Linux.

GNU ARM Eclipse Windows Build Tools v2.7-20161028* released

Oct 28, 2016 - Version 2.7-201610281058 is a maintenance release of the GNU ARM Eclipse Windows Build Tools.

GNU ARM Eclipse QEMU v2.6.0-20160728* released

Jul 28, 2016 - Version 2.6.0-201607280535 is a stable release, adding support for BASEPRI, in order for RTOSes (like FreeRTOS, µOS++ IIIe / CMSIS++) to properly implement critical sections.

GNU ARM Eclipse plug-ins v3.1.1-201606210758 released

Jun 21, 2016 - Version 3.1.1-201606210758 is a new major release; the main change is adding support for Eclipse 4.6 Neon (CDT 9.x).

GNU ARM Eclipse plug-ins v2.12.1-201604190915 released

Apr 19, 2016 - Version 2.12.1-201604190915 is a new release; the main change is the addition of a pyOCD debugging plug-in.

ARM Connected Community article, April 4, 2016

Apr 3, 2016 - A new article was published on the ARM Connected Community site, under the name GNU ARM Eclipse: more than 3.200.000 files downloaded!.

GNU ARM Eclipse plug-ins v2.11.3-201602101653 released

Feb 10, 2016 - Version 2.11.3-201602101653 is a maintenance release, intended to fix a bug introduced in 2.11.2, which, in certain conditions, prevented the template wizard to complete.
